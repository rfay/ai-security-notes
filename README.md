# AI Agent Security: Guardrails, Sandboxes, and Keeping Your Agents on a Leash

A [reveal.js](https://revealjs.com/) presentation for the Drupal AI Learners Club covering
key surfaces in AI agent security, sandboxing approaches, and the DDEV add-on ecosystem for AI tooling.

## View the Presentation

**👉 [View Live Presentation](https://rfay.github.io/ai-security-notes/)**

## Local Development

```bash
npm install
npm start
```

Opens at http://localhost:8000

## Editing

Slides are in `slides/ai-security-notes.md`. Use `---` to separate slides.

## Deploy

Push to `main` — GitHub Actions deploys to GitHub Pages automatically.
